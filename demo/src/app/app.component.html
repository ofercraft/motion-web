About Motion Library

Motion Library is a set of expressive Material 3 controls built around motion: spring-animated corner radii, variable font axes that flex with every press, and icons whose weight, grade, fill, and optical size respond to interaction. It ships in two implementations that share the same design and state model — a Jetpack Compose library for Android and framework-independent Web Components for the web.

Components

motion-button is the core control: label, icon, vertical, and icon-only layouts, four resolved states (default, pressed, selected, selected + pressed), configurable motion levels (none, low, medium, high), ripples, and haptics. motion-split-button composes two Motion buttons into Material 3 split-button geometry with shared corner morphing, per-half colors, RTL support, and animated width changes.

Feldman Font

Labels are set in Feldman Font, an OFL-licensed modified typeface based on Google Sans Flex and Fredoka. Characters supported by Google Sans Flex retain their original outlines and variable behavior; characters missing from Google Sans Flex, including Hebrew, come from Fredoka.

The Fredoka-derived glyphs were extended to respond to every included Google Sans Flex axis:

  • opsz — optical size
  • wdth — width
  • wght — weight
  • GRAD — grade, without changing advance width
  • ROND — terminal roundness
  • slnt — slant

Fredoka's original shapes are already fully rounded, so ROND=100 preserves those outlines; moving toward ROND=0 progressively flattens the outer terminal edges while preserving bowls, counters, and glyph dimensions.

Feldman Font is not affiliated with or endorsed by Google LLC or the Fredoka Project Authors. It is licensed under the SIL Open Font License 1.1; the full license and copyright notices are included in OFL.txt.

Icons

Icons are rendered with a bundled variable build of Material Symbols Rounded by Google, licensed under the Apache License 2.0.
